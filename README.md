# Azerbaijan Border Crossing Analysis (2003-2024)

## 🇬🇧 English
This project provides a comprehensive analysis and visualization of border crossing statistics in Azerbaijan. It covers data cleaning, trend analysis, and insights into migration and tourism patterns.

### 📊 Key Findings
- **Strong Correlation:** Statistical analysis reveals a strong relationship (correlation > 0.87) between citizen and foreigner border crossings.
- **Strategic Shift (2021):** For the first time, foreigner inflows surpassed citizen outflows starting from 2021, marking a structural change.

## 🇦🇿 Azərbaycan dili
Bu layihə Azərbaycanın sərhəd keçid statistikalarının analizi və vizuallaşdırılmasını əhatə edir. Layihə çərçivəsində məlumatların təmizlənməsi, trend analizi və miqrasiya/turizm axınları üzrə əsas tapıntılar qeyd edilmişdir.

### 📊 Əsas Tapıntılar
- **Güclü Korrelyasiya:** Statistik analiz vətəndaş və əcnəbi sərhəd keçidləri arasında 0.87-dən yuxarı güclü statistik əlaqə olduğunu göstərir.
- **Strateji Çevrilmə (2021):** Tarixdə ilk dəfə olaraq, 2021-ci ildən etibarən əcnəbi axını vətəndaşların xaricə səyahət axınını üstələmişdir.
- **Yeni Reallıq:** Pandemiya sonrası dövr, vətəndaş dominantlıqlı hərəkətlilikdən turizm və biznes yönümlü əcnəbi dominantlığına keçidi şərtləndirmişdir.
